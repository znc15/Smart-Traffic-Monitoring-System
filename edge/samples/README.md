# 示例素材目录

将交通监控视频或图片放入此目录，`sim_loop` 模拟模式将自动使用真实 YOLO 推理替代随机数据。

## 支持格式

- 视频：`.mp4`、`.avi`
- 图片：`.jpg`、`.png`

## 使用方式

1. 将素材文件放入 `edge/samples/` 目录
2. 以模拟模式启动边缘节点（`MODE=sim`）
3. 系统自动检测素材类型：
   - 有视频文件 → 循环播放视频，逐帧推理
   - 仅有图片 → 循环遍历图片，每张停留约 2 秒
   - 目录为空 → 回退到随机数据模式

## 建议

- 使用包含车辆的交通监控画面效果最佳
- 视频分辨率建议 640x480 或 1280x720
- 多个文件会按文件名排序后依次循环播放
