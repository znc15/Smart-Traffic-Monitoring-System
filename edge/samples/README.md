# 示例素材目录

`edge/samples/` 只在 `MODE=sim` 时使用。

当目录里有素材文件时，仿真模式会优先基于真实素材跑检测；
目录为空时，系统会回退到随机模拟数据。

## 支持格式

- 视频：`.mp4`、`.avi`、`.mov`、`.mkv`、`.webm`
- 图片：`.jpg`、`.jpeg`、`.png`、`.bmp`、`.webp`

## 使用方式

1. 把素材放到 `edge/samples/`
2. 启动 edge：

```bash
python main.py --mode sim --port 8000 --no-browser
```

3. 处理优先级：
- 有视频：循环播放视频并逐帧检测
- 只有图片：循环遍历图片并检测
- 没有素材：使用随机模拟数据

## 建议

- 优先使用固定机位交通画面，便于观察统计趋势
- 多文件会按文件名排序，建议用 `01_`、`02_` 前缀控制顺序
- 建议分辨率：`640x480` 或 `1280x720`
- 素材过大可能拖慢低配边缘设备，必要时先转码压缩
