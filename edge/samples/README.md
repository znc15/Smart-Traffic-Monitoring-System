# 示例素材目录（edge/samples）

将演示素材放入本目录后，`sim` 模式会优先使用真实素材做推理；目录为空时回退到随机模拟数据。

## 支持格式

- 视频：`.mp4`、`.avi`、`.mov`、`.mkv`、`.webm`
- 图片：`.jpg`、`.jpeg`、`.png`、`.bmp`、`.webp`

## 使用方法

1. 将素材文件放到 `edge/samples/`
2. 启动边缘节点（默认就是 `sim` 模式）
3. 系统自动按以下优先级处理：
   - 有视频：循环播放视频并逐帧检测
   - 只有图片：循环遍历图片并检测
   - 没有素材：使用随机模拟数据

## 建议

- 优先使用固定机位交通画面，便于观察趋势
- 多文件会按文件名排序，建议用 `01_`、`02_` 前缀控制顺序
- 建议分辨率：`640x480` 或 `1280x720`
- 素材过大可能拖慢低配边缘设备，必要时先转码压缩
